# 순환 소수 축약 알고리즘

순환 소수에서 순환 마디를 구하고 축약된 결과를 구합니다.



## Get 'Decimal Period'

What is [*Decimal Period ?*](https://mathworld.wolfram.com/DecimalPeriod.html)

```
1/2 = 0.5

1/7 = 0.(142857)

17/90 = 0.1(8)
```



## How to use

```
input file (or data)
---------------
4
1 2
2 1
1 7
17 90
```



```
output
---------------
#1 0.5
#2 2
#3 0.(142857)
#4 0.1(8)
```
